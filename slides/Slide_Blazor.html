🔬 Anatomía de un Componente Blazor

@page "/contador" <!-- Parte HTML/Razor --> <h3>Mi Contador</h3> <p>Valor actual: @valorActual</p> <button @onclick="Incrementar">+1</button> @code { // Parte C# (lógica) private int valorActual = 0; private void Incrementar() { valorActual++; } }

💡 Componentes clave:

  • @page "/ruta" - Define la URL
  • @valorActual - Muestra una variable
  • @onclick - Maneja clicks
  • @code { } - Tu código C#

✏️ Ejercicio Práctico

🎯 Tu turno: Crea un componente "Saludo"

Sigue estos pasos:

Bienvenido a Blazor 🔥"; } } }
